# solid-dada-icons

icon for Solidjs.

## Install

[![solid-dada-icons](https://nodei.co/npm/solid-dada-icons.png)](https://npmjs.org/package/solid-dada-icons)

## Usage

```js
import { Warning } from "solid-dada-icons";
import { render } from "solid-js/web";

const root = document.getElementById("root");

render(
  <Warning />,
  root
);
```
